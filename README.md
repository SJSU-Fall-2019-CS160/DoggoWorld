# DoggoWorld

SJSU CS160 Term Project

## Getting Started

After cloning this project, install all node dependencies

```
npm install -d
```



## Built With

* [Node.js](https://nodejs.org/en/) - The server runtime environment 
* [Express](https://expressjs.com) - Web Framework
